service | 服务,由n个**product**,按一定的线性时间结点组合而成,比如说保养,验车 order | 订单,具有相应的支付状态的服务 package | 包,由业务系统根据**order**里的服务时间结点上作优化,能合并则合并.只供管家看. service_flow | 服务流程,**service**的时间流程,线性时间结构 order_flow |
